# README #

* lib_2_5_digits_indicator

Library to control 2.5 digits 7-segment LED indicator.
Communication with indicator is done with SPI.

### Usage example ###
```C
  ... 
  Init_2_5_Digits_Indicator();
  ...
  Indicator_Display_Hex(0xF6);
  ...
```
